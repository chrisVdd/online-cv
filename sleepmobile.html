Menu

J'ai réalisé ce projet lorsque j'ai travaillé pour l'agence web Newave.

Tout a été réalisé par mes soins: création du design, intégration et développement via le CMS WordPress.

Le site est dynamique: le client peut modifier le contenu via l'interface d'administration de WordPress. Ici, deux demandes "spécifiques": la création d'un test sous forme d'un QCM, pour le test de l’échelle de somnolence d’Epworth, et la possibilité d'offir un espace privé, où seuls les médecins peuvent s'authentifier.

  • Client - Docteur Paul WULLEMAN
  • Date - août 2014
  • Compétences et outils utilisés - WordPress, jQuery (différents plugins), thème personnalisable "Bones", RSS et XML.
